# multileg_rfq_orchestrator_GUI_WIN
Multileg RFQ program with GUI for Windows

run the new install_win.bat to get the necessary dependencies for the GUI program. place the 3 new python files in the same location where you have the others. 
run python multileg_rfq_orchestrator_GUI_WIN.py from the powershell opened in the location where all .py files and the .env file are present. 
if dependencies were installed correctly this will open a user interface to edit, price and execute RFQs (no portfolio capability included yet).
